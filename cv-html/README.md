# Proyecto: Currículum Vitae en HTML Puro

Este proyecto forma parte de mi formación como **Desarrollador Full Stack Java** en **Generation Chile**. El objetivo es demostrar el dominio de la estructura semántica de HTML5, el uso de tablas para datos estructurados y el manejo de flujos de trabajo con Git/GitHub.

## 🛠️ Tecnologías utilizadas
* **HTML5**: Estructura semántica (`<header>`, `<main>`, `<section>`, `<footer>`).
* **Tablas**: Organización de historial académico.
* **Git/GitHub**: Control de versiones y despliegue.

## 📝 Aprendizajes clave
* Diferenciación entre etiquetas de bloque y de línea.
* Implementación de tablas con `<thead>` y `<tbody>`.
* Gestión de repositorios remotos mediante comandos de terminal (Git Bash)
